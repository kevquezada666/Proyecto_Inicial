# Librería de Números Complejos

Este proyecto implementa una librería en Python para realizar
operaciones con números complejos sin usar el tipo de datos
`complex` de Python.

Los números complejos se representan como tuplas:
- Forma cartesiana: (real, imaginario)
- Forma polar: (módulo, fase)

## Funciones implementadas
- Suma
- Resta
- Producto
- División
- Módulo
- Conjugado
- Conversión entre coordenadas cartesianas y polares
- Cálculo de la fase
